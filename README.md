# FedEx SCMS Logistics Analysis (2006-2015)

## 📌 Project Overview
This project analyzes the Supply Chain Management System (SCMS) of FedEx Logistics. 
The goal is to optimize delivery timelines, identify bottlenecks, and reduce 
freight costs for global health medical shipments.

## 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn, Plotly
- **Environment:** Google Colab / Jupyter Notebook

## 📊 Key Insights
- **Mode Optimization:** Shifted from high-cost Air to stable Ground transport.
- **Top Hubs:** Nigeria and South Africa represent 40% of total volume.
- **Risk Factor:** Insurance costs are more correlated to product value than weight.

## 🚀 How to Run
1. Clone this repository.
2. Open the `.ipynb` file in Google Colab.
3. Upload the `SCMS_Delivery_History_Dataset.csv` when prompted.
